|| rtsub pa daṅ ’jam bgyis pa daṅ | | ’juṅs pa rnams ni gtoṅ phod daṅ || gtum pa des (2)par
bgyis pa gaṅ | | de ni khyod kyi thabs mkhas lags |

| dga’ bo’i dbaṅ po ñer źi daṅ | | ṅa rgyal kheṅs pa ’dud pa daṅ || sor mo’i phreṅ ba’aṅ źi
bgyis na | | su źig ṅo mtshar bskyed mi bgyid |

| khyod kyi chos kyi ros tshim pas | | gser las bgyis pa’i lam spaṅs (3)nas || ran maṅ btiṅ
ba’i gnas mal du | | brten rnams bsod ñams ñal bar bgyid |

| khyod kyis dus daṅ bsam mkhyen pas | | res ’ga’ źus na’aṅ mi gsuṅ la || spros gyur gźan la
bstan pa daṅ | | druṅ du gśegs nas bśad pa’aṅ mdzad |

| dri med gos la tshon (4)bźin du | | thog mar sbyin sogs gtam dag gis || sems la spro ba bskyed
nas ni | | phyi nas chos ni bstan pa mdzad |

| ’khor ba mi bzad gtiṅ nas ni | | ’gro ba ñon moṅs draṅ slad du || khyod kyis gaṅ źig ma stsal
ba’i | | thabs daṅ spyod pa de ma mchis |

| sku yi cha lugs (5)rnam maṅ daṅ | | gsuṅ gis so so’i skad bźin du || gdul bya’i bsam pa tha
dad pa | | de daṅ der ni khyod kyis mdzad |

| de dag kun kyaṅ dag pa daṅ | | mi ’gal ba daṅ mchod ’os daṅ || brjid daṅ lha mi rnams la ni
| | sman ’dogs bkur ba śa stag lags |

| smra daṅ (6)bgyid pa phal cher ni | | legs par rdo thogs ma lags kyi || bden par gsuṅ ba khyod
la ni | | de gñis gźan du gda’ ba lags |

| bdag ñid dag pa ’ba’ źig gis | | khyod kyis ’gro ba’i dpal źiṅ mdzad || ’di ltar rnam pa ’di
’dra’i źiṅ | | ’jig rten gsum na (7)mchis ma lags |

| thog med srid pa ñam ṅa bar | | sṅa nas gcig tu ñams pa yi || sems can kun la sman slad du
| | khyod ni de ltar brtson pa lags |

| mkhas pa la bstod pa le’u bcu gñis pa’o ||
